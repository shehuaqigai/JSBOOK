定义和用法
charAt() 方法可返回指定位置的字符。
请注意，JavaScript 并没有一种有别于字符串类型的字符数据类型，
所以返回的字符是长度为 1 的字符串。
语法
stringObject.charAt(index)
参数	描述
index	必需。表示字符串中某个位置的数字
，即字符在字符串中的下标。
提示和注释
注释：字符串中第一个字符的下标是 0。
如果参数 index 不在 0 与 string.length 之间，
该方法将返回一个空字符串。
实例
在字符串 "Hello world!" 中，我们将返回位置 1 的字符：
<script type="text/javascript">

var str="Hello world!"
document.write(str.charAt(1))

</script>
