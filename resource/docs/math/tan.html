定义和用法
tan() 方法可返回一个表示某个角的正切的数字。
语法
Math.tan(x)
参数	描述
x	必需。一个以弧度表示的角。将角度乘以 0.017453293 （2PI/360）即可转换为弧度。
返回值
参数 x 的正切值。
实例
在本例中，我们将计算不同数字的正切值：
<script type="text/javascript">

document.write(Math.tan(0.50) + "
")
document.write(Math.tan(-0.50) + "
")
document.write(Math.tan(5) + "
")
document.write(Math.tan(10) + "
")
document.write(Math.tan(-5) + "
")
document.write(Math.tan(-10))

</script>
输出：
0.5463024898437905
-0.5463024898437905
-3.380515006246586
0.6483608274590866
3.380515006246586
-0.6483608274590866
