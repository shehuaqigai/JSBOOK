SVGUseElement

The SVGUseElement interface provides access to the properties of <use> elements, as well as methods to manipulate them.

Interface overview

Also implement SVGElement, SVGURIReference, SVGTests, SVGLangSpace, SVGExternalResourcesRequired, SVGStylable, SVGTransformable
Methods None

Normative document SVG 1.1 (2nd Edition)

Properties

Name Type Description
x SVGAnimatedLength Corresponds to attribute x on the given <use> element.
y SVGAnimatedLength Corresponds to attribute y on the given <use> element.
width SVGAnimatedLength Corresponds to attribute width on the given <use> element.
height SVGAnimatedLength Corresponds to attribute height on the given <use> element.
instanceRoot SVGElementInstance The root of the instance tree. See description of SVGElementInstance to learn more about the instance tree.
animatedInstanceRoot SVGElementInstance If the xlink:href attribute is being animated, contains the current animated root of the instance tree. If the xlink:href attribute is not currently being animated, contains the same value as instanceRoot. See description of SVGElementInstance to learn more about the instance tree.

Methods

The SVGUseElement interface do not provide any specific methods.
