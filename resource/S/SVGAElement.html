SVGAElement

The SVGAElement interface provides access to the properties of <a> elements, as well as methods to manipulate them.

Interface overview

Also implement SVGElement, SVGURIReference, SVGTests, SVGLangSpace, SVGExternalResourcesRequired, SVGStylable, SVGTransformable
Methods None
Properties
  • readonly SVGAnimatedString target
Normative document SVG 1.1 (2nd Edition)

Properties

Name Type Description
target SVGAnimatedString Corresponds to attribute target on the given <a> element.

Methods

The SVGAElement interface do not provide any specific methods.
