SVGFilterElement

The SVGFilterElement interface provides access to the properties of <filter> elements, as well as methods to manipulate them.

Interface overview

Also implement SVGElement, SVGURIReference, SVGLangSpace, SVGExternalResourcesRequired, SVGStylable, SVGUnitTypes
