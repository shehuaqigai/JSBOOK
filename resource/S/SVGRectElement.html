SVGRectElement

The SVGRectElement interface provides access to the properties of <rect> elements, as well as methods to manipulate them.

Interface overview

Also implement SVGElement, SVGTests, SVGLangSpace, SVGExternalResourcesRequired, SVGStylable, SVGTransformable
